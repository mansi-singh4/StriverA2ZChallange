
# Print 1 to N Using Recursion

## Question
Print numbers from `1` to `N` using recursion.

## Approach
- Start with `curr = 1`.
- Print the current number.
- Recursively call the function with `curr + 1`.
- Stop when `curr > N` (base case).

## Time Complexity
- **O(N)**

## Space Complexity
- **O(N)** (recursive call stack)






# Print Name N Times Using Recursion

## Question
Print a given name `N` times using recursion.

## Approach
- Use a counter variable starting from `0`.
- Print the name.
- Recursively call the function with `count + 1`.
- Stop when `count == N` (base case).

## Time Complexity
- **O(N)**

## Space Complexity
- **O(N)** (recursive call stack)
